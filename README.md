# TensorFlow CapsNet

## Introduction

This is the open-source implementation of [CapsNet](https://arxiv.org/pdf/1710.09829.pdf) with [TensorFlow](https://github.com/tensorflow/tensorflow).

## Usage

You can train the CapsNet model with commands.

```
./main.py
```
